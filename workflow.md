## Workflow

1. Launch OWASP ZAP and configure the browser proxy.
2. Spider the target web application to map all pages and inputs.
3. Run the Active Scan to detect vulnerabilities such as:
   - SQL Injection
   - Cross-Site Scripting (XSS)
   - CSRF
   - Security misconfigurations
4. Export the scan results.
5. Take screenshots of identified vulnerabilities.
6. Document findings and remediation notes.
